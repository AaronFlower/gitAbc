###Git Stash
gsta='git stash'          -- stash 文档
gstaa='git stash apply'   -- 恢复stash的文件 但是stash不会被删除，自己执行stash drop
gstd='git stash drop'     -- stash drop stash@{0}
gstl='git stash list'     
gstp='git stash pop'      -- 等价于 git stash apply | git stash drop
gsts='git stash show --text'
